# Gemini 网页总结助手 (Chrome Extension)

这是一个基于 Chrome 浏览器扩展程序的工具，旨在帮助用户快速提取当前网页的主要文字内容，并一键发送至 Gemini (gemini.google.com) 进行智能化总结。支持多种总结模式，包括简短总结、详细要点以及专业的 5W1H 分析。

## 🚀 主要功能

- **智能正文提取**：自动识别页面中的 `<article>` 或 `<main>` 标签，精准抓取网页核心内容。
- **多维度总结选项**：通过右键菜单提供三种预设的 AI 提示词（Prompt）：
  - **一句话简短总结**：快速获取核心大意。
  - **详细要点总结**：分点罗列文章深度内容。
  - **5W1H 分析总结**：使用 Who, What, Where, When, Why, How 模型进行结构化分析。
- **自动化操作**：自动跳转 Gemini 页面、自动填入文本、自动触发发送按钮。

## 📂 文件结构

请确保你的插件文件夹中包含以下三个文件：

1.  `manifest.json`：插件配置文件。
2.  `background.js`：后台脚本，处理右键菜单创建及网页内容抓取逻辑。
3.  `gemini_injector.js`：内容脚本，负责在 Gemini 页面自动填入和提交。

## 🛠️ 安装步骤

1.  **准备代码**：将本项目提供的三个代码块分别保存到本地文件夹（例如命名为 `GeminiSummarizer`）。
2.  **打开扩展程序页面**：在 Chrome 浏览器地址栏输入 `chrome://extensions/` 并回车。
3.  **开启开发者模式**：勾选页面右上角的“开发者模式”开关。
4.  **加载插件**：点击左上角的“加载已解压的扩展程序”，选择包含上述文件的文件夹。
5.  **确认安装**：看到“Gemini 网页总结助手”图标出现，即表示安装成功。

## 📝 使用方法

1.  打开任何包含文字内容的网页（如新闻、博客、文档）。
2.  在网页的**空白处点击右键**。
3.  找到 **“发送到 Gemini 总结当前页面”** 菜单项。
4.  根据需求选择子菜单：
    - 点击“一句话简短总结”
    - 点击“详细要点总结”
    - 点击“5W1H 分析总结”
5.  浏览器会自动打开 Gemini 标签页，并自动完成总结任务。

## ⚠️ 注意事项

- **登录状态**：使用前请确保你已经登录了 Google 账号并可以正常访问 [Gemini](https://gemini.google.com/)。
- **字符限制**：为了保证浏览器性能和 Gemini 输入框的响应速度，插件默认会截取网页的前 15,000 个字符。
- **权限说明**：插件需要 `scripting` 权限来读取当前页面的文字，以及 `storage` 权限用于在页面跳转间传递文本。

## 🔧 技术实现原理

1.  **提取阶段**：利用 `chrome.scripting.executeScript` 在当前活跃标签页执行 DOM 抓取。
2.  **存储阶段**：将抓取的文本与 Prompt 拼接后存入 `chrome.storage.local`。
3.  **注入阶段**：当检测到域名为 `gemini.google.com` 时，`gemini_injector.js` 启动。
4.  **模拟交互**：通过 `document.execCommand('insertText')` 模拟真实输入，并触发 `input` 事件激活发送按钮，最后执行自动点击。
